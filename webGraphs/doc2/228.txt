<h1 id="firstHeading" class="firstHeading" lang="en">Spring green</h1>			<div id="bodyContent" class="mw-body-content">
				<div id="siteSub" class="noprint">From Wikipedia, the free encyclopedia</div>				<div id="contentSub"><span class="mw-redirectedfrom">  (Redirected from <a href="/w/index.php?title=Dark_spring_green&amp;redirect=no" class="mw-redirect" title="Dark spring green">Dark spring green</a>)</span></div>

				<a class="mw-jump-link" href="#p-search">Jump to search</a>
				<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><div role="note" class="hatnote navigation-not-searchable">For other uses, see <a href="/wiki/Spring_Green_(disambiguation)" class="mw-redirect mw-disambig" title="Spring Green (disambiguation)">Spring Green (disambiguation)</a>.</div>
<div class="color_swatch" style="background: white; border: solid 1px black; float: right; margin: 0 .5em 1em 1.5em; font-size: 90%; text-align: center; width: 12em;">
<div style="background: #00FF7F; border-bottom: solid 1px black; color: #000000; height: 9em" title="#00FF7F"></div>
<p>Spring green (<tt>#00FF7F</tt>)<br />#00FF7F
</p>
</div>
<p><b>Spring green</b> is a <a href="/wiki/Color" title="Color">color</a> included on the <a href="/wiki/Color_wheel" title="Color wheel">color wheel</a> that is precisely halfway between <a href="/wiki/Cyan" title="Cyan">cyan</a> and green. When plotted on the <a href="/wiki/CIE_chromaticity_diagram" class="mw-redirect" title="CIE chromaticity diagram">CIE chromaticity diagram</a> it corresponds to a visual stimulus of 505 <a href="/wiki/Nanometer" class="mw-redirect" title="Nanometer">nanometers</a> on the <a href="/wiki/Visible_spectrum" title="Visible spectrum">visible spectrum</a>. <i>Spring green</i> is a pure <a href="/wiki/Munsell_color_system#Chroma" title="Munsell color system">chroma</a> on the color wheel. In <a href="/wiki/HSV_color_space" class="mw-redirect" title="HSV color space">HSV color space</a>, the expression of which is known as the <a href="/wiki/File:RBG_color_wheel.svg" title="File:RBG color wheel.svg">RGB color wheel</a>, <i>spring green</i> has a <a href="/wiki/Hue" title="Hue">hue</a> of 150°. Spring green is one of the <a href="/wiki/Tertiary_color" title="Tertiary color">tertiary colors</a> on the RGB color wheel. It is also sometimes referred to as mint green.
</p><p>The <a href="/wiki/Complementary_color" class="mw-redirect" title="Complementary color">complementary color</a> of <i>spring green</i> is <a href="/wiki/Rose_(color)" title="Rose (color)">rose</a>.
</p>

<p><b>Aquamarine</b> is a color that is a pale bright <a href="/wiki/Tint" class="mw-redirect" title="Tint">tint</a> of spring green toned toward cyan. It represents the color of the <a href="/wiki/Aquamarine_(gemstone)" class="mw-redirect" title="Aquamarine (gemstone)">aquamarine</a> gemstone.
</p>

<p><b>Persian green</b>  is a <a href="/wiki/Color" title="Color">color</a> used in <a href="/wiki/Pottery" title="Pottery">pottery</a> and <a href="/wiki/Persian_carpet" title="Persian carpet">Persian carpets</a> in <a href="/wiki/Iran" title="Iran">Iran</a>.
</p><p>Other colors associated with Persia include <a href="/wiki/Persian_red" title="Persian red">Persian red</a> and <a href="/wiki/Persian_blue" title="Persian blue">Persian blue</a>. The color <i>Persian green</i> is named from the green color of some Persian pottery and  is a representation of the color of the mineral <a href="/wiki/Malachite" title="Malachite">malachite</a>.  It is a popular color in <a href="/wiki/Iran" title="Iran">Iran</a> because the color green symbolizes gardens, nature, heaven, and sanctity. The first recorded use of <i>Persian green</i> as a color name in English was in 1892.<sup id="cite_ref-18" class="reference"><a href="#cite_note-18">&#91;18&#93;</a></sup><sup id="cite_ref-19" class="reference"><a href="#cite_note-19">&#91;19&#93;</a></sup>
</p>

<p><b>Sea green</b> is a shade of <a href="/wiki/Cyan" title="Cyan">cyan</a> color that resembles the <a href="/wiki/Sea_floor" class="mw-redirect" title="Sea floor">sea floor</a> as seen from the surface.
</p><p><i>Sea green</i> is notable for being the emblematic color of the <a href="/wiki/Levellers" title="Levellers">Levellers</a> party in the politics of 1640s England. Leveller supporters would wear a sea-green ribbon, in a similar manner to the present-day red <a href="/wiki/AIDS" class="mw-redirect" title="AIDS">AIDS</a> awareness ribbon. 
</p>
<div class="thumb tleft"><div class="thumbinner" style="width:252px;"><a href="/wiki/File:Flag_of_the_British_Republic.svg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c3/Flag_of_the_British_Republic.svg/250px-Flag_of_the_British_Republic.svg.png" width="250" height="125" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c3/Flag_of_the_British_Republic.svg/375px-Flag_of_the_British_Republic.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c3/Flag_of_the_British_Republic.svg/500px-Flag_of_the_British_Republic.svg.png 2x" data-file-width="600" data-file-height="300" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/File:Flag_of_the_British_Republic.svg" class="internal" title="Enlarge"></a></div>Flag of a proposed "<a href="/wiki/Republicanism_in_the_United_Kingdom" title="Republicanism in the United Kingdom">British Republic</a>" used by <a href="/wiki/Chartists" class="mw-redirect" title="Chartists">Chartists</a> and Radicals in the nineteenth century. Sea green colors were often used by the Chartists and earlier <a href="/wiki/Levellers" title="Levellers">Levellers</a>. A group of 'English republican' intellectuals used a version of this tricolor with blue at the top.</div></div></div>

<p><b>Pine green</b> is a rich dark shade of spring green that resembles the color of <a href="/wiki/Pine_tree" class="mw-redirect" title="Pine tree">pine trees</a>. It is an official Crayola color (since 1903) that is this exact shade in the Crayola crayon, but in the markers, it is known as <i><a href="/wiki/Crocodile" title="Crocodile">crocodile</a> green</i>.
</p>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/File:Bhutan_pine_tree.JPG" class="image"><img alt="Pine tree in Bhutan" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/f6/Bhutan_pine_tree.JPG/220px-Bhutan_pine_tree.JPG" width="220" height="293" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/f6/Bhutan_pine_tree.JPG/330px-Bhutan_pine_tree.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/f6/Bhutan_pine_tree.JPG/440px-Bhutan_pine_tree.JPG 2x" data-file-width="1944" data-file-height="2592" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/File:Bhutan_pine_tree.JPG" class="internal" title="Enlarge"></a></div><a href="/wiki/Pine" title="Pine">Pine</a> tree in <a href="/wiki/Bhutan" title="Bhutan">Bhutan</a></div></div></div>
<p>The color <i>pine green</i> is a representation of the average color of the <a href="/wiki/Leaf" title="Leaf">leaves</a> of the trees of a <a href="/wiki/Coniferous_forest" class="mw-redirect" title="Coniferous forest">coniferous forest</a>. The color <i>pine green</i> was originally known as <i>pine tree</i>.  The first recorded use of <i>pine tree</i> as a color name in English was in 1923.<sup id="cite_ref-21" class="reference"><a href="#cite_note-21">&#91;21&#93;</a></sup>
</p>
