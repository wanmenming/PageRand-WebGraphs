<p><b>Green S</b> is a green synthetic <a href="/wiki/Coal_tar" title="Coal tar">coal tar</a> triarylmethane <a href="/wiki/Dye" title="Dye">dye</a> with the molecular formula C<sub>27</sub>H<sub>25</sub>N<sub>2</sub>O<sub>7</sub>S<sub>2</sub>Na.
</p><p>As a <a href="/wiki/Food_dye" class="mw-redirect" title="Food dye">food dye</a>, it has <a href="/wiki/E_number" title="E number">E number</a> E142. It can be used in <a href="/wiki/Mint_sauce" title="Mint sauce">mint sauce</a>, <a href="/wiki/Dessert" title="Dessert">desserts</a>, <a href="/wiki/Gravy" title="Gravy">gravy</a> granules, <a href="/wiki/Sweets" class="mw-redirect" title="Sweets">sweets</a>, <a href="/wiki/Ice_cream" title="Ice cream">ice creams</a>, and tinned <a href="/wiki/Pea" title="Pea">peas</a>.  Green S is prohibited as a food additive in <a href="/wiki/Canada" title="Canada">Canada</a>, <a href="/wiki/United_States" title="United States">United States</a>, <a href="/wiki/Japan" title="Japan">Japan</a>, and <a href="/wiki/Norway" title="Norway">Norway</a>.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">&#91;<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (November 2008)">citation needed</span></a></i>&#93;</sup> It is approved for use as a <a href="/wiki/Food_additive" title="Food additive">food additive</a> in the EU<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> and Australia and New Zealand.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup>
</p><p>Green S is a <a href="/wiki/Vital_dye" class="mw-redirect" title="Vital dye">vital dye</a>, meaning it can be used to <a href="/wiki/Staining_(biology)" class="mw-redirect" title="Staining (biology)">stain</a> living <a href="/wiki/Cell_(biology)" title="Cell (biology)">cells</a>. It is used in <a href="/wiki/Ophthalmology" title="Ophthalmology">ophthalmology</a>, along with <a href="/wiki/Fluorescein" title="Fluorescein">fluorescein</a> and <a href="/wiki/Rose_bengal" title="Rose bengal">rose bengal</a>, to diagnose various disorders of the eye's surface, dry eyes for example.
</p>
