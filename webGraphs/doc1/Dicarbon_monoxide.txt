<p><b>Dicarbon monoxide</b> (C<sub>2</sub>O) is a <a href="/wiki/Molecule" title="Molecule">molecule</a> that contains two <a href="/wiki/Carbon" title="Carbon">carbon</a> <a href="/wiki/Atom" title="Atom">atoms</a> and one <a href="/wiki/Oxygen" title="Oxygen">oxygen</a> atom. It is a linear molecule that, because of its simplicity, is of interest in a variety of areas.  It is, however, so extremely <a href="/wiki/Reactivity_(chemistry)" title="Reactivity (chemistry)">reactive</a> that it is not encountered in everyday life.  It is classified as a <a href="/wiki/Cumulene" title="Cumulene">cumulene</a> and an <a href="/wiki/Oxocarbon" title="Oxocarbon">oxocarbon</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>
</p>
<h2><span class="mw-headline" id="Occurrence">Occurrence</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Dicarbon_monoxide&amp;action=edit&amp;section=1" title="Edit section: Occurrence">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Dicarbon monoxide is a product of the <a href="/wiki/Photodissociation" title="Photodissociation">photolysis</a> of <a href="/wiki/Carbon_suboxide" title="Carbon suboxide">carbon suboxide</a>:<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup><sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup>
</p>
<dl><dd>C<sub>3</sub>O<sub>2</sub>  → CO  + C<sub>2</sub>O</dd></dl>
<p>It is stable enough to observe reactions with <a href="/wiki/Nitric_oxide" title="Nitric oxide">NO</a> and <a href="/wiki/Nitrogen_dioxide" title="Nitrogen dioxide">NO<sub>2</sub></a>.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup>
</p><p>Called <b>ketenylidene</b> in <a href="/wiki/Organometallic_chemistry" title="Organometallic chemistry">organometallic chemistry</a>, it is a <a href="/wiki/Ligand" title="Ligand">ligand</a> observed in <a href="/wiki/Metal_carbonyl_cluster" title="Metal carbonyl cluster">metal carbonyl clusters</a>, e.g. [OC<sub>2</sub>Co<sub>3</sub>(CO)<sub>9</sub>]<sup>+</sup>.  Ketenylidenes are proposed as intermediates in the chain growth mechanism of the <a href="/wiki/Fischer-Tropsch_Process" class="mw-redirect" title="Fischer-Tropsch Process">Fischer-Tropsch Process</a>, which converts <a href="/wiki/Carbon_monoxide" title="Carbon monoxide">carbon monoxide</a> and <a href="/wiki/Hydrogen" title="Hydrogen">hydrogen</a> to hydrocarbon fuels.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup>
</p><p>The <a href="/wiki/Organophosphorus_compound" title="Organophosphorus compound">organophosphorus compound</a> (C<sub>6</sub>H<sub>5</sub>)<sub>3</sub>PCCO (CAS# 15596-07-3) contains the C<sub>2</sub>O functionality.  Sometimes called Bestmann's Ylide, it is a yellow solid.<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">&#91;6&#93;</a></sup>
</p>
